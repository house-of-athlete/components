## Component usage

See compositions for examples of styling links and using render props.

### External Link

```js live
<CMSLink link={{ _type: 'externalLink', url: 'http://example.com' }}>
  External Link
</CMSLink>
```

### Internal Link

```js live
<CMSLink
  link={{
    _key: 'd3d49d559627',
    _type: 'internalLink',
    document: {
      _type: 'collection',
      path: { _type: 'slug', current: '/apparel' },
    },
  }}
>
  Internal Link
</CMSLink>
```
